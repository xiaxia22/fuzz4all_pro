# Class SkeletonNotFoundException

### Class Description

```java
@Deprecated

public class
SkeletonNotFoundException

RemoteException
```

A

SkeletonNotFoundException

is thrown if the

Skeleton

corresponding to the remote object being
exported is not found. Skeletons are no longer required, so this
exception is never thrown.

# Class InvalidRequestStateException

**Source:** `jdk.jdi\com\sun\jdi\request\InvalidRequestStateException.html`

### Class Description

```java
public class
InvalidRequestStateException

extends
RuntimeException
```

Thrown to indicate that the requested event cannot be modified
because it is enabled. Filters can be added only to disabled
event requests.
Also thrown if an operation is attempted on a deleted request.
See

EventRequestManager.deleteEventRequest(EventRequest)

**All Implemented Interfaces:** Serializable

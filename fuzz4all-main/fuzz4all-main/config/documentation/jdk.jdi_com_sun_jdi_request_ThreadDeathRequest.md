# Interface ThreadDeathRequest

**Source:** `jdk.jdi\com\sun\jdi\request\ThreadDeathRequest.html`

### Class Description

```java
public interface
ThreadDeathRequest

extends
EventRequest
```

Request for notification when a thread terminates in the target VM.
When an enabled ThreadDeathRequest is satisfied, an

event set

containing a

ThreadDeathEvent

will be placed on the

EventQueue

.
The collection of existing ThreadDeathRequests is
managed by the

EventRequestManager

**All Superinterfaces:** EventRequest

,

Mirror

# Interface CSSFontFaceRule

**Source:** `jdk.xml.dom\org\w3c\dom\css\CSSFontFaceRule.html`

### Class Description

```java
public interface
CSSFontFaceRule

extends
CSSRule
```

The

CSSFontFaceRule

interface represents a @font-face rule in
a CSS style sheet. The

@font-face

rule is used to hold a set
of font descriptions.

See also the

Document Object Model (DOM) Level 2 Style Specification

.

# Interface AudioClip

**Source:** `java.desktop\java\applet\AudioClip.html`

### Class Description

```java
@Deprecated
(
since
="9")
public interface
AudioClip
```

The

AudioClip

interface is a simple abstraction for
playing a sound clip. Multiple

AudioClip

items can be
playing at the same time, and the resulting sound is mixed
together to produce a composite.

**Since:** 1.0

### Method Details

#### void play()

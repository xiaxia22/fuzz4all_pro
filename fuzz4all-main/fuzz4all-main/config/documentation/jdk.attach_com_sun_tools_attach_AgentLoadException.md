# Class AgentLoadException

**Source:** `jdk.attach\com\sun\tools\attach\AgentLoadException.html`

### Class Description

```java
public class
AgentLoadException

extends
Exception
```

The exception thrown when an agent cannot be loaded into the target
Java virtual machine.

This exception is thrown by

VirtualMachine.loadAgent

or

VirtualMachine.loadAgentLibrary

,

loadAgentPath

methods
if the agent, or agent library, cannot be loaded.

# Class JSException

**Source:** `jdk.jsobject\netscape\javascript\JSException.html`

### Class Description

```java
public class
JSException

extends
RuntimeException
```

Thrown when an exception is raised in the JavaScript engine. This is merely
a marker class to indicate an exception relating to the JavaScript
interface.

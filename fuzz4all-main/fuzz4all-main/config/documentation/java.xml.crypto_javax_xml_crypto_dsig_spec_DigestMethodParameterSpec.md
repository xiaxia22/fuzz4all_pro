# Interface DigestMethodParameterSpec

**Source:** `java.xml.crypto\javax\xml\crypto\dsig\spec\DigestMethodParameterSpec.html`

### Class Description

```java
public interface
DigestMethodParameterSpec

extends
AlgorithmParameterSpec
```

A specification of algorithm parameters for a

DigestMethod

algorithm. The purpose of this interface is to group (and provide type
safety for) all digest method parameter specifications. All digest method
parameter specifications must implement this interface.

**All Superinterfaces:** AlgorithmParameterSpec
